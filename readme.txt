== Description ==

Automatically Insert and Optimize Google Adsense code to your site for increase your profits.
Here are some of the reasons why this plugin is fantastic:

* Help you to create the AdSense Privacy Policy page

* Helps you to reduce the risk of ban from the Google program.

* Add ads in content, but also in sidebar, footer, header ….

* Not more than 3 ads per page to meet google policy

* Multi Language Supported (English, Español, Italiano)

* Number of ads can be dependent on the post length.

* ads widget

* Intelligent insertion without breaking lists, pictures etc.

* Comfortable definition of ad size, layout and colors in admin panel.

* different channels for all ads to control and optimize different styles, colors etc.



== Installation ==

* Download

* Extract and copy Adsense Extreme folder to Plugins Directory (/wp-content/plugins) of your blog. (do not rename!) or you can install it from within your wordpress admin panel to do this click on add new under the plugin tab then search for Adsense Extreme you will see " Adsense Extreme " plugin by Mario Gentile the Install Now link this will automatically install the plugin

* Activate Plugin

* To know how to Make settings go to http://www.adsenseplugin.it/

* Get rich
